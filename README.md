# Talks with Polly

AWS Polly を使った音声合成結果の検証用アプリ

## AWS Polly Configuration

以下の環境変数を設定しておく必要があります。

- ENV['AWS_ACCESS_KEY_ID']
- ENV['AWS_SECRET_ACCESS_KEY']
- ENV['AWS_REGION']
